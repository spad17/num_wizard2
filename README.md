# 🤖 NumWizard Bot

Telegram-бот для гема `num_wizard2`.

## 🚀 Запуск

```bash
# 1. Клонировать
git clone https://github.com/spad17/num_wizard2.git
cd num_wizard_bot

# 2. Установить зависимости
bundle install

# 3. Настроить токен
cp .env.example .env
# Открой .env и вставь токен от @BotFather

# 4. Запустить
ruby lib/num_wizard_bot.rb
